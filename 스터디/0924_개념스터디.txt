[인공지능(AI)의 이해]
인공지능의 개념
 : 현재까지의 개발 방법론 => 결과를 유도하는 식을 만드는 방식
 : 인공지능의 개념 => 현재까지의 개발 방법론과 다르게 결과를 보고 식을 기계(컴퓨터)가 유추하도록 유도

-> 머신러닝의 개념 : 퍼셉트론
 : 데이터와 값을 확인하고 이에 따라 식을 유도하는 방식으로, Experience(데이터)와 Task(목적)에 따라 Perfomance를 유도

-> 딥러닝의 개념
 : 머신러닝의 한계 : XOR문제를 해결하기 위해 도입한 멀티 레이어 퍼셉트론에 따라
=> 현재까지는 데이터와 목적을 위해 2차원 적인 가설(Hypothesis)를 유도하는 단순 학습의 형태에서
=> 멀티 레이어와 활성화 함수, 은닉층을 통해  다차원 적인 가설을 유도하도록 개념의 확장

: 결국 평가를 위한 지표(metrics)를 통해(손실함수 공식이 발생) Performance를 개선해 나가는 알고리즘.
-> 손실함수의 개념(역전파), 활성화함수(머신러닝의 한계를 다차원으로 해결하기 위함)

[리눅스(Linux)]
 : 과거 최초의 컴퓨터 애니악에 사용하던 C언어를 훨씬 평이하고 용이하게 사용하기 위한 운영체제 UNIX를
리눅스 발토즈에 의해 모든 사람이 컴퓨터를 사용할 수 있도록 제안된 UNIX기반 운영체제로
"컴퓨터는 모든 사람을 위한 것 이다"라는 신념 하 개발

리눅스의 모토는 한 프로그램에 여러 기능을 담지 않는 것으로 명령어의 수가 60여가지로 구성되어있으며
현재의 MSA(마이크로 서비스 아키텍쳐)와 맥락이 동일함.

서버로 활용하는 가장 큰 이유는 UNIX 자체가 에니악과 데이터를 주고 받으며 개발된 운영체제로 
서버용으로 개발되어 가볍고 빠른 모습을 확인할 수 있음.
=> 개발자들 사이에서는 감자만 있어도 리눅스는 활용이 가능하다고 할 정도

우리가 활용할 줄 알아야 하는 내용은 본즈 언어로 작동하는 Bash와 쉘 스크립트 sh. => 도커 등에 사용하는 언어
